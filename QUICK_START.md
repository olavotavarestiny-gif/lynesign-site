## 🎨 Customize in 30 Seconds

### Change Colors
Edit `app/globals.css`:
```css
--primary-gold: #YOUR_COLOR;
--accent-coral: #YOUR_COLOR;
```

### Change Text
Edit component files, e.g., `app/components/HeroSection.tsx`:
```tsx
<h1>Your New Headline</h1>
```
